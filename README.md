# Retail Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes store sales data using Power BI to generate insights about sales performance, profit trends, and customer purchasing behavior.

The dashboard helps in understanding key business metrics and supports data-driven decision making.

---

## Tools Used
• Power BI
• Microsoft Excel
• Data Visualization
• Data Cleaning

---

## Dataset
The dataset contains store sales transaction details including:
• Order Date
• Sales Amount
• Profit
• Category
• Region
• Customer Segment

---

## Dashboard Features
• Sales trend over time
• Profit analysis
• Category-wise performance
• Regional sales distribution
• Key performance indicators (KPIs)

---

## Key Insights
• Identified top performing product categories
• Observed monthly sales trends
• Compared regional performance
• Analyzed profit distribution
