# Retail Sales Analytics Dashboard

## Project Overview

This project is a **Power BI dashboard** created to analyze retail sales data.  
The dashboard helps to understand sales performance, revenue trends, product categories, customer behavior, profit analysis, and overall business growth.

The main goal of this project is to convert raw retail sales data into meaningful visual insights using Power BI.

---

## Project File

- **Project Name:** Retail Sales Analytics Dashboard
- **Tool Used:** Microsoft Power BI
- **File Type:** `.pbix`
- **Dashboard Type:** Data Analytics / Business Intelligence Dashboard

---

## Objectives

The main objectives of this project are:

- To analyze overall retail sales performance.
- To calculate total sales, profit, and quantity sold.
- To identify top-performing products and categories.
- To understand customer purchasing behavior.
- To analyze region-wise or city-wise sales performance.
- To study monthly sales and profit trends.
- To create an interactive dashboard for business decision-making.

## Data Cleaning and Transformation

The dataset was cleaned and transformed using **Power Query** in Power BI.

Major steps included:

- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Formatted date columns.
- Created useful calculated columns.
- Prepared the dataset for dashboard visualization.
